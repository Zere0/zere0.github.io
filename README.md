## 🚀 Getting Started

### Prerequisites
- Ruby (2.6 or higher)
- Bundler

### Installation
```bash
# Clone the repository
git clone https://github.com/zere0/zere0.github.io.git
cd zere0.github.io

# Install dependencies
bundle install

# Start the development server
bundle exec jekyll serve
```

### Customization
1. **Colors**: Modify CSS variables in `assets/css/custom.css`
2. **Content**: Update `_config.yml` for site settings
3. **Posts**: Add new posts in `_posts/` directory
4. **Pages**: Create new pages in `_tabs/` directory
